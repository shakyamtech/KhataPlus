# KhataPlus

KhataPlus is a POS (Point of Sale), stock, and ledger management platform designed for small retail and grocery shops to manage their daily business operations.

## Features

- **POS Billing:** Quick invoice and bill generation with change calculation.
- **Inventory Management:** Track stock levels, pricing, categories, and low stock alerts.
- **Party Ledger:** Manage customer credit (Udhaar) and supplier transactions.
- **Cashbook:** Record daily cash inflows and outflows.
- **Reports:** View sales, purchases, and profit reports.
- **Balance Sheet:** View a summary of assets, liabilities, and capital.
- **Recipe System:** Auto-deduct stock for items assembled in-house.
- **Multilingual Support:** Supports English and Nepali languages.

## Getting Started

### Prerequisites

You need Node.js and npm installed.

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
